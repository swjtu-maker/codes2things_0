制作手册

  1. 概述、造物时代、项目管理
    • Github pages项目网站制作简明教程（强烈推荐）
    • 阅读：Github Helloword Guidehelloword github，建立项目仓库
    • 阅读：知乎上的github教程：Github 四步走
    • 阅读使用automatic generator （也可以这样：Github pages）使用githubpages建立网站
    • 安装Download & Install Github Desktop ，然后将网站同步到本地目录，下一步可以使用编辑器来编辑网站了。如果github desktop在线安装失败，可以尝试这里离线下载
    • 更多教程：Using github
    • 视频教程：版本控制入门 – 搬进 Github开始学习
  2. 计算机辅助设计
    • Fusion 360入门培训(PPT)
    • Fusion 360教程(美工刀)
    • Fusion 360视频教程(台灯)
    • Fusion 360视频教程(螺丝刀)
    • Fusion 360视频教程(吊灯灯罩)
    • Fusion 360视频教程(Arduino盒子)
  3. 切割
    • 下载CorelDRAW软件安装包到自己的电脑，依照 CorelDRAW软件安装教程 或者信邦激光4060软件安装-雕刻-切割操作视频教程 在自己的电脑安装软件。
    • 参考 CorelDRAW激光雕刻教程（官方）和信邦激光4060软件安装-雕刻-切割操作视频教程，学习CorelDRAW软件的使用方法。
    • 利用第二节课程《计算机辅助设计》所学知识绘图，将图像导入CorelDRAW，打开激光雕刻机，参考 激光雕刻机操作规程雕刻自己的作品。
    • BoxMaker盒子设计工具 是一个激光切割盒子设计软件，大家可以在这里快速简单的设计自己所需要尺寸的矩形盒子。
    • 纸艺大师Pepakura Designer 纸艺大师（Pepakura Designer），是一款从3D模型作成展开图的软件，只按一个按钮即可作成展开图，再用各种纸编辑功能，可设计”好看、有趣、容易作”的展开图。 。
